# Commit Strategy

Use conventional, descriptive commits:

```text
feat(parser): add tree-sitter extraction pipeline
feat(storage): implement sqlite metadata layer
feat(retrieval): add hybrid lexical retrieval
feat(patterns): implement pattern extraction engine
feat(mcp): add MCP server interface
docs(architecture): add retrieval planner design
```

Avoid:

```text
fix
update
new feature
test
wip
final
```
